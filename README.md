# 4:1 Multiplexer Verification using Cocotb

## Overview
This project implements a 4:1 Multiplexer in Verilog and verifies it using Cocotb.

## Tools Used
- Verilog
- Verilator
- Cocotb
- GTKWave
- Python

## Files
- mux4.v
- tb_mux4.v
- test_mux.py
- Makefile

## Running Simulation

```bash
make
```
